# Base Rules

## MUST
1. 文件编码必须为 UTF-8（无 BOM）。
2. 输出语言默认使用中文。
3. 避免中文乱码或编码转义文本。
4. 行尾统一为 LF（`\n`）。

## 输出前检查清单
1. 是否为 UTF-8 无 BOM。
2. 是否不存在 CRLF。
3. 中文内容是否可直接阅读。

## 代码要求
1. 所有代码都必须包含详细的注释，包括函数、类、模块等，通用方法必须使用jsdoc注释。
2. 注释必须使用中文。
3. 注释必须与代码保持一致。
